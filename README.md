# vue-todolist

[Vue.js]를 학습하고 TodoList 코드 작성 및 리팩토링하기

1.options API
2.Composition API - setup()
3.Composition API - <script setup> 활용해서 리팩토링하기

4. 3번에 스타일링하기

기능

1. todolist 입력
2. todolist 시각화
3. todolist 삭제
4. tab기능 - 할일/완료된일

추가기능

- 수정하기 : 텔레포트의 modal로 구현하기
- localDB활용해서 저장하기
- 디자인 하기
